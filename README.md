# Career Recommendation System

## Description
A JavaFX-based Career Recommendation System that suggests degree paths or professions based on user skills, academics, and interests.

## Technologies Used
- Java 17
- JavaFX
- Maven
- MySQL
- JDBC

## Modules
- Authentication Module
- Skills Module
- Academic Module
- Questionnaire Module
- Recommendation Engine
- Result Module

## How to Run
1. Clone the repository
2. Open in IntelliJ / Eclipse
3. Configure MySQL database
4. Run Main.java

## Database
- MySQL
- JDBC Connectivity
